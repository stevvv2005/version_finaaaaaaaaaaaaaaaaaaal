# CityFlow

Structure du projet:

- `index.html`: structure HTML de l'application.
- `assets/css/styles.css`: styles CSS.
- `assets/js/app.js`: logique JavaScript et controles de l'interface.
- `data/`: donnees separees utilisees par l'application.

Lancer en local:

```bash
node .codex-static-server.js
```
